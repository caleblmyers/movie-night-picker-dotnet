# Error Log

Recurring issues and their solutions. When you hit a bug that took real debugging effort, log the root cause here so it doesn't waste time again.

## Template

```markdown
### YYYY-MM-DD — Short description

**Symptom:** What you observed
**Root cause:** Why it happened
**Fix:** What resolved it
**Prevention:** How to avoid it in the future (if applicable)
```
